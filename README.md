easy-build
==========

Perform an easy build of the GENIVI Yocto Baseline using Docker

System Requirements
-------------------

* Docker 0.9.1 or later
* A fast Internet connection

Building
--------

```
docker build github.com/gmacario/easy-build
```

Running
-------


```
docker run -i -t github.com/gmacario/easy-build
```
